# Mindloop - Техническая документация

## Обзор проекта
Mindloop — это современная монохромная лендинг-платформа для рассылок и контента. Сайт выполнен в минималистичном стиле с использованием глубокого черного цвета и акцентов на типографику и видео-контент.

## Технологический стек
- **Framework**: Next.js (React)
- **Язык**: TypeScript
- **Стилизация**: Tailwind CSS
- **Компоненты**: shadcn/ui
- **Анимации**: Framer Motion
- **Видео-стриминг**: hls.js

## Руководство по стилю
### Цветовая палитра (Monochrome)
- **Background**: #000000 (Pure Black)
- **Foreground**: #FFFFFF (Pure White)
- **Accent**: HSL(170, 15%, 45%) - Тонкий акцентный цвет
- **Liquid Glass**: Полупрозрачные элементы с размытием заднего плана (backdrop-filter: blur(4px)).

### Типографика
- **Основной шрифт**: Inter (Sans-serif) - используется для основного текста и навигации.
- **Акцентный шрифт**: Instrument Serif (Serif) - используется для курсивных выделений в заголовках.

## Компоненты
### Navbar
- Фиксированная позиция, полностью прозрачный фон.
- Содержит логотип (концентрические круги), навигационные ссылки и социальные иконки в стиле liquid-glass.

### Hero Section
- Полноэкранное видео с автовоспроизведением.
- Плавный градиент внизу для перехода к контенту.
- Форма подписки с эффектом liquid-glass.

### Grid Gallery
- Адаптивная сетка 4x4.
- Динамическая загрузка данных из внешнего API через переменные окружения.
- Обработка состояний загрузки и ошибок.

### Mission Section
- Скролл-анимация: постепенное появление слов (opacity transition) при прокрутке.

### CTA Section
- Фоновое видео, транслируемое через HLS (hls.js) для оптимальной производительности.

## Инструкции по разработке
### Установка зависимостей
```bash
npm install
```

### Запуск в режиме разработки
```bash
npm run dev
```

### Переменные окружения
Создайте файл `.env` на основе `.env.example`:
- `NEXT_PUBLIC_API_URL`: URL API для загрузки карточек.
- `NEXT_PUBLIC_API_KEY`: Ключ доступа к API (если требуется).

## Поддержка
Для добавления новых секций используйте вспомогательную функцию `fadeUp` для обеспечения единообразия анимаций появления контента.
